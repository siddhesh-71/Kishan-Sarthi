
**An AI-Powered Digital Companion for Farmers**

KishanSarthi is an integrated, farmer-centric platform designed to empower Indian farmers with **data-driven insights, government information, and modern agricultural tools**.
The platform bridges the gap between technology and grassroots farming by combining **AI, web technologies, and domain knowledge** into a single accessible system.

---

## 🚜 Problem Statement

Indian farmers face multiple challenges such as:

* Unpredictable crop yields
* Lack of access to real-time agricultural guidance
* Difficulty understanding government schemes and policies
* Limited decision-making support for crop planning

Most existing solutions are fragmented and not tailored to farmers’ real needs.

---

## 💡 Solution Overview

**KishanSarthi** acts as a **digital guide (Sarthi)** for farmers by providing:

* Crop prediction and guidance using AI
* Centralized access to agricultural schemes and policies
* Market and community insights
* Simple and user-friendly web interface

The goal is to **increase productivity, reduce uncertainty, and improve farmers’ livelihoods**.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/RIT2006ESH/KishanSarthi.git
cd KishanSarthi
```

---

### 2️⃣ Frontend Setup

Open `frontend/index.html` directly in a browser
or use a live server extension.

---

### 3️⃣ Backend Setup

Navigate to the backend directory:
```bash
cd crop-prediction-backend
```

Install Node.js dependencies:
```bash
npm install
```

Install Python dependencies:
```bash
pip install -r requirements.txt
```

Start the server:
```bash
npm start
```
The server will run on `http://localhost:5000`.
